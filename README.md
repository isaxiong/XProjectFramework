# XProjectFramework
基于MVVM架构搭配ARouter搭建的项目框架

## 一、说明
#### 本项目主要基于MVVM + ARouter搭建整体的框架，在新项目的时候方便移植并根据需求做定制化修改，提升开发速度


## 二、引用的依赖库

- **框架：ARouter + MVVM + DataBinding**
- **网络：RxEasyHttp（Retrofit+okHttp）**
- **异步：RxJava2 + RxAndroid2**
- **通信：EventBus/RxBus**
- **RV：  BRVAH**
- **图片：Glide**
- **存储：GreenDao + MMKV**
- **日志：XLog**
- **适配：autoSize**
- **推送：友盟/信鸽**
- **打包：walle**
- **埋点：根据埋点规则来，是否能单独抽取出一个定制库**
